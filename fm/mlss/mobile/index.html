Mobile demo

This demo is 320px wide, which is the width of the iPhone SE.

With additional padding, the width would be even smaller: FM's component (.iw_component) is 290px, and FM's article (.article-wrapper.padding-30) leaves 230px.

Responsive demo

This demo is responsive -- it adjusts to browser width. The maximum content width is 783px, which is maximum FM column width (843px) minus 30px borders. It's easy to limit the chart to a smaller maximum.
