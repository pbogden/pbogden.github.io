Mobile demo

The smallest mobile devices (e.g., iPhone 5 and SE) have a screen width of 320px, which is the width of this paragraph.

This demo has no borders. By default, the FM website puts 45px borders on each side. So if the borders cannot be removed, content will need to be 230px wide.

Responsive demo

In this demo, the container for the graphic is responsive. Regardless of browser size, the graphic fills the container, which has a maximum width of 783px. That's the maximum content width on the FM website -- maximum column width is 843px, with 30px of padding on all sides. Adjust the width of your browser to see the effect.
